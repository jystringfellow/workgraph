# Agent: Resume Context

## Purpose

Reconstruct working context for a project.

## Input

- project name

## Output

- recent events
- relevant files
- related people
- open tasks, if known
- suggested next step

## Requirements

- Prioritize recency
- Include evidence from events, files, or memory
- Avoid speculation without evidence
- Keep output structured enough to test

## Notes

This is a core UX feature. It should make it easy to restart work without rereading everything manually.
